# Bug Report: Required radio option is missing on product page

## Environment
Website: https://demo.opencart.com  
Browser: Google Chrome  
OS: Windows 11

## Preconditions
User opens a product page that contains required options.

## Steps to Reproduce
1. Open the website https://demo.opencart.com
2. Navigate to a product page (example: Apple Cinema 30").
3. Scroll to the Available Options section.
4. Try to add the product to the cart.

## Expected Result
The user should see radio button options and be able to select one of them before adding the product to the cart.

## Actual Result
The system shows the message "Radio required", but no radio button options are displayed.  
As a result, the user cannot select an option and cannot add the product to the cart.

## Severity
High

## Priority
High

## Notes
Missing required options block the purchasing process and prevent the user from completing the action.
