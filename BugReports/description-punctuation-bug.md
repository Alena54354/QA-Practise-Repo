# Bug Report: Incorrect punctuation in device description

## Environment
Manufacturing inspection interface (AOI system)  
Device type: Optical transceiver  
Testing stage: Visual inspection

## Preconditions
The operator opens the AOI inspection interface to review device information after testing.

## Steps to Reproduce
1. Open the AOI inspection interface.
2. Navigate to the device information section.
3. Locate the description field for the optical transceiver.
4. Observe the formatting of the description text.

## Expected Result
The device description should be clearly formatted and easy to read, without misleading punctuation.

## Actual Result
Incorrect punctuation in the description makes the text appear similar to a phone number or link, which may confuse the operator.

## Severity
Low

## Priority
Low

## Notes
Although this issue does not affect the functionality of the system, it may reduce readability and create confusion during device inspection.
