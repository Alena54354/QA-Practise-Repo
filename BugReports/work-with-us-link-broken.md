# Bug Report: "Work With Us" button leads to 404 page

## Environment
Website: https://www.ryzesuperfoods.com  
Browser: Google Chrome  
OS: Windows 11

## Preconditions
User is browsing the Ryze Superfoods website.

## Steps to Reproduce
1. Open the website https://www.ryzesuperfoods.com
2. Navigate to the Our Story page.
3. Scroll down to the section where the Work With Us button appears.
4. Click Work With Us.

## Expected Result
The user should be redirected to a page with job opportunities or company career information.

## Actual Result
The user is redirected to a 404 Page Not Found.

## Severity
Medium

## Priority
Medium

## Notes
Broken navigation links negatively affect user experience and prevent users from accessing important site sections.
